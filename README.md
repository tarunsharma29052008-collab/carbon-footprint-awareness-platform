# Carbon Footprint Awareness Platform

A Streamlit-based application to calculate and visualize carbon footprint.

## Features

- Carbon Footprint Calculator
- Eco Score
- Tree Offset Calculator
- Pie Chart Visualization
- Green Tips
- Bar Chart
- AI Recommandation
- Eco- Friendly Tips

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
