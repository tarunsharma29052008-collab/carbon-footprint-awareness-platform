# 🌍 Carbon Footprint Awareness Platform

A smart and interactive Streamlit application that helps users calculate, visualize, and reduce their carbon footprint through personalized recommendations and sustainability insights.

---

## 📖 Problem Statement

Climate change is one of the biggest global challenges today. Many people are unaware of how their daily activities contribute to carbon emissions.

This platform helps users:

- Measure their carbon footprint
- Understand emission sources
- Set reduction goals
- Track sustainability progress
- Receive eco-friendly recommendations

---

## 🚀 Features

### 🌱 Carbon Footprint Calculator
Calculate emissions based on:

- Transportation usage
- Electricity consumption

### 📊 Interactive Dashboard

- Real-time calculations
- KPI metrics
- Progress tracking

### 📈 Data Visualization

- Bar Chart
- Pie Chart
- Yearly Trend Analysis

### 🤖 AI Recommendations

Personalized suggestions based on user emissions.

### 🎯 Goal Tracking

Users can set carbon reduction targets and monitor progress.

### 🌳 Tree Offset Calculator

Calculates how many trees are required to offset generated emissions.

### 📜 History Tracking

Stores previous calculations during the session.

### 📥 Download Reports

Generate and download carbon footprint reports.

### ♿ Accessibility Features

- Help tooltips
- Expandable information sections
- User-friendly interface

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/carbon-footprint-awareness-platform.git
```

### Move into Project Folder

```bash
cd carbon-footprint-awareness-platform
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
